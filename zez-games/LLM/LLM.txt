# -ngl 99 	GPU Offload. Sends 99 layers to GPU.        Ensures 100% of the model runs on your 16GB VRAM.
#
# -fa         Flash Attention.                            Crucial. Speeds up RDNA 4 and saves VRAM on long chats.
#
# -c 16384    Context Size. 	                            16k is a "sweet spot" for 16GB cards. 8k is faster, 32k might squeeze VRAM.
#
# -b 512 	    Batch Size.                                 Lower (512) feels more "snappy" for chatting; higher (2048) is for summarizing long docs.
#
# --mlock     Memory Lock.                                Prevents Linux from moving the model to "Swap" (slow SSD/HDD).
#
# --mmap      Memory Mapping. 	                        Allows the model to load almost instantly from disk.

# -cnv        ChatGPT Style Chat
